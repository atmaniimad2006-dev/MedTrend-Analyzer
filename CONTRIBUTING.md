# Contributing to MedTrend-Analyzer

Thank you for your interest in contributing to the MedTrend-Analyzer project!

## How to Contribute

1. **Fork the Repository**: Start by forking the repository to your own GitHub account.
2. **Clone Locally**: Clone your fork to your local machine.
3. **Create a Branch**: Create a feature branch (`git checkout -b feature/amazing-feature`).
4. **Make Changes**: Implement your feature or bug fix.
5. **Run Tests**: Ensure the project compiles and tests pass using Maven.
   ```bash
   mvn clean install
   ```
6. **Commit**: Commit your changes with descriptive messages.
7. **Push & Pull Request**: Push to your branch and submit a Pull Request to the `main` branch.

## Code of Conduct
Please ensure a welcoming and inclusive environment for everyone.
